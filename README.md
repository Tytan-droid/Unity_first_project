## Game Overview
This project is a **Metroidvania** game where players explore a large interconnected world, unlock new abilities, and backtrack to access previously unreachable areas.

The game focuses on:
- Exploration  
- Platforming  
- Combat  
- Level design  

Each new power unlocks new paths, encouraging players to revisit earlier zones and discover hidden secrets.

---

## Key Features

- 2D side-scrolling gameplay  
- Platforming challenges  
- Enemies and combat system  
- Collectibles and secrets
